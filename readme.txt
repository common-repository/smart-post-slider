== Description ==
Smart  post slider is a super responsive and very advance wordpress slider . You can easily control this slider.Here you'll get  many option to control your slider.It lets you add your latest blog post.
You can easily use this slider. 

<strong>Features:-</strong>
<ul>
 	<li>Slide your latest post</li>
 	<li>You can shown or hide title</li>
 	<li>Setup sliding speed</li>
 	<li>Setup the number of sliding item from dashboard</li>
 	<li>You can playing single or multiple slide</li>
 	<li>You can change background color from dashboard</li>
 	<li>You can change image height form dashboard</li>
 	<li>You can true/false pagination,pagination number, and navigation</li>
 	<li>and more feature</li>
</ul>

  How to use this plugin:-
 After active plugin, just copy and past this "[post_slider category="Uncategorized" count="10"]" shortcode of your any post or paste, then see the magic.
